Workflow anatomy

This workflow was created and run in the CLI guide.

name: hello-world
description: Minimal workflow that logs a message.

steps:
  - id: greeting
    action: transform
    with:
      template: "Hello from Weavr at {{ currentISODate }}"

  - id: log-it
    action: log
    needs: [greeting]
    with:
      message: "Message: {{ steps.greeting }}"

Verified actions

These actions were exercised during the verification runs.

Action Purpose
transform Creates a value by interpolating a template string.
log Prints a message to the run output.
