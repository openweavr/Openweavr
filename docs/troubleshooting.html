TTY initialization failed

Observed error: ERR_TTY_INIT_FAILED.

Use non-interactive flags for weavr onboard, weavr create, and weavr ask. 
node weavr.mjs onboard --non-interactive --ai-provider none --skip-web-search
node weavr.mjs create --non-interactive --name docs-demo --trigger manual
node weavr.mjs ask "Create a workflow that logs hi" --non-interactive --output /tmp/weavr-ask.yaml

Gateway listen errors

Observed error: listen EPERM in restricted environments.

Run the gateway in a normal terminal session with permission to bind to local ports.

Workflow directory mismatch

The CLI uses ~/.weavr/workflows while the gateway defaults to workflows/ in its current working directory.

Set workflowsDir in ~/.weavr/config.yaml if you want the gateway to read the same workflows as the CLI.

Doctor warnings

The doctor command can warn when no plugins are installed. Built-in plugins still load.
