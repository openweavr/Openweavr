Prerequisites

Verified with Node.js 22.22.0 on February 2, 2026.

Use Node.js 22 or newer to match the build settings in the project.

Install and build

Run these commands from the project root.

npm install
npm run build
npm run build:web

Runtime paths

These are the filesystem locations used by the CLI and gateway.

Path Purpose
~/.weavr/config.yaml CLI and gateway configuration.
~/.weavr/workflows/ Workflow files used by CLI commands.
workflows/ Default gateway workflow directory (relative to server CWD).
dist/ Backend and CLI build output.
dist/web/ Web UI build output served by the gateway.
