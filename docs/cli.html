Configure without a TTY

This path avoids interactive prompts and works in non-TTY shells.

node weavr.mjs onboard --non-interactive --ai-provider none --skip-web-search

Create a workflow

Generate a starter workflow file from flags.

node weavr.mjs create --non-interactive \
  --name docs-demo \
  --description "Docs demo workflow" \
  --trigger manual

You can also create the YAML manually:

cat <<'EOF' > ~/.weavr/workflows/hello.yaml
name: hello-world
description: Minimal workflow that logs a message.

steps:
  - id: greeting
    action: transform
    with:
      template: "Hello from Weavr at {{ currentISODate }}"

  - id: log-it
    action: log
    needs: [greeting]
    with:
      message: "Message: {{ steps.greeting }}"
EOF

List workflows

node weavr.mjs list

Run a workflow

node weavr.mjs run hello

Generate a workflow with AI

Non-interactive mode can write YAML to a file.

node weavr.mjs ask "Create a workflow that logs hi" \
  --non-interactive \
  --output /tmp/weavr-ask.yaml

Doctor

node weavr.mjs doctor
