Self-hosted workflow automation with AI agents

Weavr is an open-source workflow automation platform that combines the power of traditional automation tools with native AI agent capabilities. Run it on your own infrastructure with full data sovereignty.

Open Source Self-Hosted AI-Native Node.js 22+

Quick Start

Get up and running in under a minute.

# Clone and install
git clone https://github.com/openweavr/Openweavr.git
cd Openweavr
npm install

# Build backend and web UI
npm run build
npm run build:web

# Start the server
node weavr.mjs serve

Open http://localhost:3847 to access the web interface.

Example Workflow

A simple AI-powered workflow that responds to GitHub issues.

name: issue-responder
description: Auto-respond to new GitHub issues with AI analysis

trigger:
  type: github.issue.opened
  with:
    repo: your-org/your-repo

steps:
  - id: analyze
    action: ai.agent
    with:
      task: |
        Analyze this GitHub issue and provide a helpful initial response:

        Title: {{ trigger.issue.title }}
        Body: {{ trigger.issue.body }}

        Be concise and helpful. If it's a bug, ask for reproduction steps.
        If it's a feature request, acknowledge it and ask clarifying questions.

  - id: respond
    action: github.create_comment
    needs: [analyze]
    with:
      repo: your-org/your-repo
      issue_number: "{{ trigger.issue.number }}"
      body: "{{ steps.analyze.response }}"
