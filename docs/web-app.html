Web app

Start the gateway Verify UI and config Workflows and runs API

Start the gateway

Serve the web UI and the API from a single process.

node weavr.mjs serve --port 3851 --host 127.0.0.1
Open http://127.0.0.1:3851 in your browser after the gateway starts.

Verify the UI and config endpoint

curl -I http://127.0.0.1:3851/
curl -s http://127.0.0.1:3851/api/config/status

Create and run a workflow via API

The Workflows and Runs screens use these endpoints.

cat <<'EOF' > /tmp/weavr-web-verified.yaml
name: web-verified
description: Workflow created via API for docs verification.

steps:
  - id: greeting
    action: transform
    with:
      template: "Hello from API at {{ currentISODate }}"

  - id: log-it
    action: log
    needs: [greeting]
    with:
      message: "Message: {{ steps.greeting }}"
EOF

python3 - <<'PY'
import json
from pathlib import Path
payload = {
  "name": "web-verified",
  "yaml": Path("/tmp/weavr-web-verified.yaml").read_text()
}
Path("/tmp/weavr-web-verified.json").write_text(json.dumps(payload))
PY

curl -s -X POST http://127.0.0.1:3851/api/workflows \
  -H "Content-Type: application/json" \
  --data-binary @/tmp/weavr-web-verified.json

curl -s http://127.0.0.1:3851/api/workflows

curl -s -X POST http://127.0.0.1:3851/api/workflows/web-verified/run \
  -H "Content-Type: application/json" \
  -d '{}'

curl -s http://127.0.0.1:3851/api/runs
